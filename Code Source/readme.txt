Ce jeu est une version Beta du jeu "Elvis dans le monde des morts" 

Pour pouvoir y jouer :
1-Install Eclipse
2-File-> Open Project From File system
3-Select your project 
4-Select the game package from the Package Explorer 
5-Click right on the game package and click run as
6-Select JavaApplication


La fenetre menu du jeu s'ouvrira 
Sélectionez votre niveau de jeu 

Pour pouvoir déplacer le héros les touches !

8:haut 
7:haut gauche 
9:haut droit 
4:gauche
1:bas gauche
2:bas
3:bas droit 
6:droit

Il est recommandé d'utiliser un clavier comportant un pavé numérique pour une meilleur expérience de jeu 

A toi de jouer!
